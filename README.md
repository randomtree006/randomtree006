## 关于数学学习的一些想法

颗粒感

1.每周日的下午及晚上是math free时间，这段时间除了做数学什么都可以做

2.设置数分一小时，每天在数学分析上进行一小时独立思考的知识的输出，也许周日一上午做别的没时间数分一小时，但周六大概会远超一小时，所以没关系的

3.这个寒假注定每天要与数学相伴了，目前打算在过年后开学前的时间出去旅游，理由如下

（1）如果过年前进行旅游，尤其如果一放寒假就旅游，怕心散了，回来之后若是无法专注数学，将是无法挽回的损失

（2）这个时间段的旅游可以同时承担多个作用，第一可以逃避走亲戚，第二可以在一个假期的数学学习和即将开学之间获得喘息，第三过年期间本身不可避免会放松，旅游也是放松，与其分成两截，不如捏在一起，自己也不会有过年期间还要学数学的委屈自己的感觉

（3）人不可能一直连轴转的，旅游期间可以保持每天低强度的数学
